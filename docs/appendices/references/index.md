## Standards and Protocols

### Communication Standards
- DDS (Data Distribution Service): Real-time communication middleware
- CAN Bus: Controller Area Network for embedded systems
- EtherCAT: Real-time Ethernet communication

### Safety Standards
- ISO 13482: Safety requirements for personal care robots
- ISO 12100: Safety of machinery
- IEC 62061: Functional safety of machinery

## Tools and Libraries

### Perception Libraries
- OpenCV: Computer vision and machine learning software library
- PCL: Point Cloud Library for 2D/3D image and point cloud processing
- ROS Perception: Collection of perception packages for ROS

### Control Libraries
- ROS Control: Generic control framework for ROS
- RT-Tools: Real-time tools for robotics applications
- Control Toolbox: Collection of control algorithms
