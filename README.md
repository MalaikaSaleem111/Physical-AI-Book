# Physical AI & Humanoid Robotics Book

This repository contains a comprehensive book covering Physical AI and humanoid robotics fundamentals, designed for university students, early researchers, and beginners in the field.

## Overview

This book provides clear, structured content about:
- Physical AI and embodied intelligence fundamentals
- Humanoid robotics principles
- ROS 2 concepts and architecture
- Simulation environments (Gazebo, Unity)
- NVIDIA Isaac platform
- Vision-Language-Action (VLA) systems
- Weekly learning modules for systematic progression

## Features

- **Theory-First Approach**: All examples are conceptual, not executable
- **University-Level Content**: Comprehensive explanations suitable for academic study
- **Structured Learning Path**: 8-12 chapters with weekly modules
- **Docusaurus Powered**: Clean, navigable documentation site

## Weekly Learning Modules

The book is structured around 13 weekly modules covering:
- Weeks 1-3: Foundations in Physical AI and humanoid robotics
- Weeks 4-6: Development tools including ROS 2 and simulation
- Weeks 7-9: Advanced systems and integration
- Weeks 10-12: Applications and future directions

## Requirements

- No hardware or software setup required to understand the content
- Basic programming knowledge recommended
- Access to the deployed documentation site
